# 📝 BÀI TẬP VỀ NHÀ - CSS CƠ BẢN

## Chủ đề

Thiết kế **Personal Card** bằng HTML và CSS.

## Yêu cầu

Tạo **2 file**:

- `index.html`
- `style.css`

Liên kết CSS bằng **External CSS**.

### Giao diện gồm:

- Ảnh đại diện.
- Họ và tên.
- Nghề nghiệp.
- Thông tin cá nhân (Địa chỉ, Trường học, Sở thích).
- 3 kỹ năng.
- 2 nút: **Follow** và **Message**.

## Hướng dẫn

Trong bài làm, hãy áp dụng các kiến thức CSS đã học:

- Reset CSS (`margin`, `padding`, `box-sizing`).
- `background`, `color`.
- `font-family`, `font-size`, `font-weight`.
- `text-align`, `text-transform`.
- `width`, `height`.
- `border`, `border-radius`.
- `padding`, `margin`.
- `display: block` và `display: inline-block`.
- Sử dụng các đơn vị: `px`, `%`, `em`, `rem`.

## Lưu ý

- Giao diện rõ ràng, cân đối.
- Card được căn giữa trang.
- Không sử dụng **Flexbox**, **Grid** hoặc thư viện CSS (Bootstrap, Tailwind,...).
- Khuyến khích thêm màu sắc để giao diện đẹp hơn.
